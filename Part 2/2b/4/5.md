# Классы проектирования

- GameLoop как Mediator
- MainMenuState / PlayingState / PauseState / GameOverState . Паттерн State, состояния игры, в разных состояниях поведение приложения и допустимые команды разные
- SwapCommand / UseBonusCommand / ExitCommand etc. Паттерн Command для инкапсуляции запросов на действия
- BombBonus / ClearColorBonus : IFieldBonusEffect . Паттерн Strategy для эффектов бонусов
- Field : IObservable . Паттерн Observer для передачи изменений игрового поля на отрисовку, в статистику, лог и т.д.
