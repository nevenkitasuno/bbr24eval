# Кластеры

Из оставшихся классов удалось выделить следующие кластеры

## Кластер управления приложением

- GameLoop
- MainMenu, PauseMenu
- MainMenuState / PlayingState / PauseState / GameOverState

## Кластер ввода

- Control
- SwapCommand / UseBonusCommand / ExitCommand / etc.

## Кластер игровой логики

- Field
- BombBonus / ClearColorBonus : IFieldBonusEffect
- Field : IObservable
- Матрица для хранения элементов Field
- Словарь позиций элементов, которые будут удалены после очередного витка игровой логики
