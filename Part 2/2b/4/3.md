# Классы анализа

- Ball: элемент поля
- Field: матрица 8x8 из Ball

- Stats: информация о статистике текущей партии. Может создавать Bonus

- GameLoop: оркестратор. Принимает события, отправляет их получателям, вызывает отрисовку. Содержит вспомогательные методы объединяющие игровые события и ведение статистики
- EventBase: событие для GameLoop
- CommandEvent: содержит команду от игрока

- BonusBase: базовый класс для бонусов. Можно передать его классам, поведение которых он изменяет
- Cords: координаты Field, например их может вводить игрок
- MenuBase: базовый класс для меню
- MainMenu : MenuBase: главное меню
- PauseMenu : MenuBase: меню паузы
- CommandBase: базовый класс для команды, полученной от игрока
- Control: преобразует консольный ввод в команды и отправляет их в изменяемые классы
