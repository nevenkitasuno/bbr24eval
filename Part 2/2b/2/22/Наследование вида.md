Класс Transport может содержать следующие классы в качестве полей (либо одну из осей можно представить с помощью наследования):
- Класс PowerSource с подклассами GasPowered, ElectricPowered, HybridPowered
- Класс MovementType с подклассами GroundTransport, AirTransport, WaterTransport

Никакой из этих критериев не главнее, и они активно комбинируются друг с другом.
